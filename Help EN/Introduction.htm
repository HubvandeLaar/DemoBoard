Introduction

Since quite some time many chess trainings are based upon the Dutch ‘Stappen Methode’, a profound method with instructor materials and booklets with exercises for the pupils. The booklets contain diagrams with pieces marked with a rectangle or circle. There are several chess-programs suited for usage with a digiboard, but none with rectangles. Also the PGN-file does not provide for black dots, circles and rectangles. See also : Setup Toolbar

You are welcome to use this application as intended. The application is designed to be used by chess trainers.

  • As a digitial demo board
  • With quick setup using the keyboard
  • Show games of children (so strange moves can also be stored)
  • Lessons can be saved as XPGN, PGN or CPS
  • Lessons can also be saved as PDF (for printout)
