Training Question

Questions are stored at the 'Comment-before' with the new question format of Chessbase ("[%tqu "Question"... ]). When the 'Mode' is set to 'Training'and a game contains a question, the moves unto the question are played and a form popups with the question.

This question request the user to perform the correct move at the board.

When the answer is correct, feedback is given and the score is increased with the specified number of points. Other options can provide other feedback and another amount points. For incorrect answers, message to retry is shown, and the board is restored to the original position. For answers detailed in a subvariant of the correct move, these subvariant moves can be played using the 'Details' button
