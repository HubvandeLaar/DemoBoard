Setup Toolbar

Shows the pieces, and graphical elements at the right-side of the chessboard.

Not all graphical elements are supported by all file formats. When these elements are not supported they will not be present while opening the file. You can always add these elements. When these elements are not supported they are omitted when exporting to that format. So creating a diagram with an ‘Star’ and saving it as PGN-file will omit the star.
