Quiz

Provide the correct move

Questions at the ‘Comment-before’ with the new question format of Chessbase (“[%tqu “Question”….. ]). If this question is present, the Quiz ‘View’ automatically popups with the question when moving to the next move.

These questions request the user to perform the correct move at the board.

When the answer is correct, feedback is given and the score increase with the specified number of points. Other options can provide other feedback and another amount points. For answers not specified, a message to retry is shown, and the board is restored to the original state.
