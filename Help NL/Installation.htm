Installatie

Bij gebruik met een digibord is installatie niet nodig.
Gebruik dan de shortcut op de USB-stick om de application te starten.

Bij installatie op de harde geschijf; kopieer de folder DemoBoard van USB-stick naar een plaats op de harde schijf. Maak een shortcut aan die wijst naar ‘DemoBoard.exe’.

Of start altijd op vanaf de USB-stick.
